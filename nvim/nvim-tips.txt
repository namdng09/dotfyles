* Marks
ma         " set mark 'a' at the current cursor location
`a         " go to exact position of mark 'a'
'a         " go to the beginning of the line of mark 'a'
''	Jump back to the position before the last jump
````	Jump back to the exact position before the last jump
"[	Beginning of last changed or pasted text
"]	End of last changed or pasted text
'.	Last edit position
'^	Position before last insert mode
''	Position before last jump
'"	Last exited position in a file (very useful)
- delete specific marks: :delmarks a, multiple at once: :delmarks a b c
- delete marks use a range: :delmarks a-z
